Showing latest tweets with new Twitter API v1.1

== Description ==

Since the Twitter updated it's API most of old Twitter widgets don't work. Now Twitter requires new authentication, so you have to create
[new Twitter application](https://dev.twitter.com/apps/ "Twitter Dev Center") and use 4 keys. Don't blame me, it's how Twitter decide...

Live demo you can see in footer [of our demo site](http://demo.brankic.net/bigbangwp/twitter-page/ "Our demo site")


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload bra_twitter_widget.zip to plugins, like any other plugin, or upload unzipped folder bra_twitter_widget to wp-content/plugins/
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Widgets and use Brankic Twitter Widget in sidebar you want
1. Create [new Twitter application](https://dev.twitter.com/apps/ "Twitter Dev Center") and use 4 keys.
